# Hypoc - 程序测试与验证系统

基于程序测试与验证工具，支持输出比对、源码格式检查和字符集检测。

## 功能特性

### 输出检查
- **预设模式**：选择问题后自动使用内置测试数据进行比对
- **自定义模式**：上传 Demo 程序和用户程序进行对比
  - 支持上传 checkdata 文件
  - 支持在线构造检查点

### 源码检查
- **格式检查**：基于规则检查 C/C++ 代码风格
- **字符集检查**：检测 GB2312 编码，支持编码转换

### 支持的问题列表
3-b5, 3-b7, 3-b8, 3-b9, 3-b10, 3-b12-1, 3-b13-1, 3-b13-2,
4-b1, 4-b2, 4-b3, 4-b5, 4-b6, 4-b7, 4-b8, 4-b9, 4-b10, 4-b11, 4-b12, 4-b13, 4-b14,
5-b8, 5-b9, 5-b10, 5-b15, 5-b16, 5-b17, 5-b18,
6-b1, 6-b2, 6-b3

## 运行

```bash
python main.py
```

## 项目结构

```
Hypoc/
├── main.py              # 主程序入口
├── core/
│   ├── comparison.py    # 比对逻辑
│   └── checkdata.py     # checkdata 解析
├── ui/
│   ├── tabs.py         # Tab 页面
│   └── cards.py         # 卡片组件
├── Rule/
│   └── style_checker.py # 代码风格检查
└── char/
    ├── check_gb2312.py      # GB2312 检测
    └── convert_to_gb2312.py # GB2312 转换
```

## 依赖

- Python 3.8+
- PyQt6 >= 6.4.0
